LPL的比赛打了有段时间，电视TV端的应用缺不支持4k，羸弱的性能也不能运行一些特殊的软件，只能通过特殊的办法进行投屏。

### 作用：

使用 [real-url](https://github.com/wbt5/real-url) 提供的函数/算法 获取指定的直播间链接并跳转。

### 准备

1. 安装依赖库

```shell
pip install -r requirements.txt
```

2. 启动服务

```shell
python3 web.py
```
